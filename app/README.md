### Prerequisites
Git client

Go - 1.7 or later (for releases before v1.0, 1.6 or later)

npm(v4.1.2 or later) and node(v6.10.0 or later)

### Deploy


Install redis server and start it
```
git clone https://github.com/abhilekhsingh041992/realtime-chat
cd app
npm install
npm start bin/www
```
